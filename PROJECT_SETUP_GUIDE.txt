QUICK START
================================================================================

1. Run setup script:
   python setup.py

2. Move files to correct locations:
   - Your CSV → data/
   - Your method_categories → config/method_categories.json
   - Your method_shortnames → config/method_shortnames.json
   - Your method_variant_groups → config/method_variants.json

3. Update config.py:
   - Set CSV_PATH to your CSV filename
   - Adjust TOPIC_INDICES if needed

4. Install dependencies:
   pip install -r requirements.txt

5. Run dashboard:
   python dashboard.py

6. Open browser:
   http://localhost:8050
